🚀 CodeAlpha Internship Tasks
Welcome to my repository! This contains the web development projects I successfully completed during my Full Stack Development Internship at CodeAlpha.

Both projects are built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) following industry-standard practices, clean architecture, and modern UI/UX design using Tailwind CSS.

📂 Projects Included
1. 🛒 CodeAlpha E-Store (E-Commerce Platform)
A complete, production-ready e-commerce web application featuring a seamless shopping experience from browsing products to placing orders.

📂 Path: CodeAlpha-E-Store

🛠️ Tech Stack: React.js, Tailwind CSS, Node.js, Express.js, MongoDB (Mongoose), JWT, Context API.

✨ Key Features:

Product Management: Dynamic product listings, detailed product pages, and search/category filtering.
Shopping Cart: Global state management using React Context API (Add, remove, update quantity without page reload).
Authentication & Security: Secure user registration/login with JWT tokens and encrypted passwords (bcryptjs).
Order Processing: Complete checkout system that saves customer details and orders directly to the database.
Protected Routing: Secure checkout and orders pages accessible only to logged-in users.
Responsive Design: Fully mobile-friendly and optimized UI.
2. 💬 CodeAlpha Social App (Mini Social Media)
A dynamic mini social media application featuring core social networking functionalities like posting, following, and real-time interactions.

📂 Path: CodeAlpha-Social_App

🛠️ Tech Stack: React.js, Tailwind CSS, Node.js, Express.js, MongoDB (Mongoose), JWT, React Router.

✨ Key Features:

User Profiles: Dynamic profile pages accessible via usernames, featuring user bios and post history.
Edit Profile: Users can update their name and bio in real-time.
Post Feed: A smart feed algorithm that shows posts from the user and the people they follow.
Like/Unlike System: Toggle-based like system with real-time count updates.
Comment System: Users can add comments on posts instantly.
Follow/Unfollow System: Users can follow others, with dynamic follower/following counts.
Search Users: A robust search bar to find other users by name or username.
⚙️ How To Run These Projects
Since both projects follow the standard MERN architecture, the setup is identical for both:

Clone the repository:
git clone https://github.com/YOUR_USERNAME/CodeAlpha-tasks.git
Navigate to the project:
bash

cd CodeAlpha-E-Store  # or CodeAlpha-Social_App
Setup Backend:
Go to the backend folder: cd backend
Install dependencies: npm install
Create a .env file and add your MONGO_URI and JWT_SECRET.
Start the server: npm run dev (Runs on port 5000)
Setup Frontend:
Go to the frontend folder: cd frontend
Install dependencies: npm install
Start the app: npm run dev (Runs on port 5173)
